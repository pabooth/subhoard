# Support

## Supported versions

Only the latest released version is supported and receives security fixes.
Users should reproduce problems on the latest release before reporting them.

## Getting help

- Use a [bug report](https://github.com/pabooth/subhoard/issues/new/choose) for
  reproducible defects.
- Use a
  [feature request](https://github.com/pabooth/subhoard/issues/new/choose) for
  proposed improvements.
- Use
  [private vulnerability reporting](https://github.com/pabooth/subhoard/security/advisories/new)
  for security issues.

This is a community-maintained open-source project. Support is provided on a
best-effort basis with no response-time or resolution-time guarantee.

Before opening an issue, remove cookies, SMTP credentials, subscriber content,
private publication data, generated archives, and other sensitive data.

## Scope

Support covers the documented reference configuration and current release.
General Substack, Camoufox, SMTP-provider, operating-system, browser-session,
and network administration questions may need to be handled through those
projects' support channels.
